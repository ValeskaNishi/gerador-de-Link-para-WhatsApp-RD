# Gerador de Link para WhatsApp — RD Station

Ferramenta para gerar links de WhatsApp com mensagem automática, desenvolvida como parte do desafio técnico da RD Station.

## Tecnologias

- Next.js 15
- React 19
- TypeScript 5
- Tailwind CSS 3
- React Hook Form 7

## Pré-requisitos

- Node.js 18 ou superior
- npm

## Como rodar

1. Clone o repositório:

```bash
git clone https://github.com/ValeskaNishi/gerador-de-Link-para-WhatsApp-RD.git
```

2. Entre na pasta do projeto:

```bash
cd gerador-de-Link-para-WhatsApp-RD/
```

3. Instale as dependências:

```bash
npm install
```

4. Build o projeto:

```bash
npm run build
```

5. Rode a aplicação:

```bash
npm run dev
```

A aplicação estará disponível em: http://localhost:3000
